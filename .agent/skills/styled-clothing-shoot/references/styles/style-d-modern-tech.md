# 风格 D：现代科技

本文件定义 `现代科技 (D)` 的独立风格说明。目标是更适合女装表达的未来都市时装感，而不是空泛科幻背景。

## 风格定位

- 核心气质：冷调、精致、未来感、干净利落
- 视觉重点：高级商场、美术馆、玻璃展厅、金属廊道、亚克力装置、镜面材质、柔光灯带
- 应避免：云端、天空岛、星空地面、廉价赛博霓虹、纯 CG 概念背景

## 与相邻风格边界

- 不要写成 `艺术建筑 (E)`：现代科技偏冷调玻璃和金属，不是暖米色矿物墙。
- 不要写成 `都市松弛 (C)`：现代科技更时装大片，不是生活方式空间。
- 不要写成 `室外街拍 (F)`：现代科技尽量在受控室内或半室内空间完成。

## 关键词与影调

- 关键词：玻璃展厅、灯带、银灰、亚克力、金属框架、镜面地面、展台、未来都市
- 色盘：银灰、冷白、雾蓝、雾粉、低饱和金属色
- 光线：冷白主光、线性灯带、反射光、低饱和硬朗轮廓

## 场景家族

| 家族 | 推荐 micro-location | 画面特征 | 更适合的镜头 |
|---|---|---|---|

## 20 镜头编排建议

- 前 1-4 张至少覆盖玻璃 / 金属 / 亚克力 3 种材质
- 不要把 20 张都写成同一条灯带走廊
- 坐姿 hero 更适合镜面台阶、冷白展台、金属长凳
- 近景要明确金属、玻璃、亚克力、反光地面或灯带线条，不要只剩冷色背景

## 批量避重池

- 轮换轴：材质（玻璃 / 金属 / 亚克力 / 镜面）、光源（冷白均匀光 / 灯带 / 反射 / 顶光）、空间尺度（长廊 / 大厅 / 小型装置区）
- 高危重复骨架：
  - `金属长廊正面 + 金属长廊背面 + 灯带长廊近景`
  - `全部都在商场扶梯口，缺少展厅与装置变化`

## 负面约束

- 不要生成紫蓝赛博朋克夜景
- 不要出现概念云海、星空、全息漂浮文字
- 不要把现代科技写成普通办公楼或家电卖场
- 不要让背景材质缺失，只剩纯灰色棚拍墙
