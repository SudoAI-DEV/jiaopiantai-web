# 风格 A：海边艺术

本文件定义 `海边艺术 (A)` 的独立风格说明。`styled-clothing-shoot` 仍以 `SKILL.md` 中的 20 镜头规则为准；本文件只负责帮助 Agent 把海景写得更丰富，避免退化成“空沙滩 + 海平线”的单一游客照。

## 风格定位

- 核心气质：自由、度假、带一点艺术感的海滨大片
- 视觉重点：岩石海岸、礁石、潮汐池、巴厘岛式热带海边、海边餐厅、海边艺术馆、棕榈屋、海岸步道
- 应避免：空沙滩游客照、廉价海边打卡点、单一蓝天海平线、只有一堵白墙的伪海景

## 与相邻风格边界

- 不要写成 `室外街拍 (F)`：海边艺术不是城市滨海商业街，也不是门店外摆抓拍。
- 不要写成 `田园自然 (B)`：海边艺术的植物应服务于热带度假或海边建筑，不是花园庄园。
- 不要写成 `大自然 (G)`：海边艺术重点仍然是海岸叙事，不是山野旅拍。

## 关键词与影调

- 关键词：岩石海岸、潮汐池、海边艺术馆、热带棕榈屋、海边餐厅、海风、礁石、海岸步道、日落前暖光
- 色盘：海蓝、礁石灰、奶油白、暖金、热带绿
- 光线：优先清晨明亮阳光、午后侧逆光、黄金时刻暖光、海面反射补光

## 20 镜头编排建议

- 前 1-4 张全身图至少覆盖：岩石海岸 / 热带建筑 / 海岸步道 3 个家族
- 第 5-8 张不要继续重复“站在海边看海”，要切换到栏杆、台阶、门廊、坐姿、回身等结构性镜头
- 第 9-10 张 hero 图允许环境占比更大，但服装仍是第一视觉中心
- 近景不要把海景糊成纯蓝背景；哪怕是细节图，也要让礁石、水面、棕榈叶、栏杆或建筑边缘保持可辨识

## 批量避重池

- 优先轮换轴：海岸类型（礁石 / 潮汐池 / 步道 / 建筑露台）、建筑类型（棕榈屋 / 餐厅 / 艺术馆）、时间段（清晨 / 午后 / 黄金时刻）
- 高危重复骨架：
  - `空沙滩正面站姿 + 海平线背面`
  - `同一块礁石反复拍正反两张`
  - `20 张都在蓝海背景前，没有热带建筑或器物变化`

## 负面约束

- 不要生成拥挤游客、比基尼度假群众、海滩躺椅密集区
- 不要把背景简化成纯沙滩 + 纯海平线
- 不要出现廉价海边婚纱照式 pose
- 不要把海边艺术馆写成现代科技展厅
