# 风格 C：都市松弛

本文件定义 `都市松弛 (C)` 的独立风格说明。重点是高端生活方式空间里的自然状态，不是纯街头抓拍。

## 风格定位

- 核心气质：城市生活感、室内外连通、轻松但不邋遢
- 视觉重点：咖啡馆、公寓、画廊、精品酒店公区、艺术馆咖啡区、设计空间
- 应避免：拥挤街景、普通办公室、廉价网红店内景、纯棚拍白底空间

## 与相邻风格边界

- 不要写成 `室外街拍 (F)`：都市松弛更偏室内或内外过渡，不是纯街边抓拍。
- 不要写成 `现代科技 (D)`：都市松弛可以有设计空间，但不是冷硬科技展厅。
- 不要写成 `田园自然 (B)`：绿植可以出现，但只是空间点缀，不是花园主角。

## 关键词与影调

- 关键词：咖啡馆、公寓客厅、设计家具、画廊、书架、酒店公区、窗边、杂志、松弛
- 色盘：米白、浅灰、暖木、深咖、奶油色、少量低饱和跳色
- 光线：室内窗光、午后自然光、柔和反射光、低对比暖调

## 20 镜头编排建议

- 前 1-4 张至少覆盖咖啡馆 / 公寓 / 画廊 3 类空间
- 中段全身图要主动加入坐姿、靠门、窗边、走廊动态，避免连续站姿
- 近景要让书页、咖啡杯、边桌、窗框、挂画或家具线条成为环境锚点
- 拼图适合把面料、口袋、包带、杯具与桌面关系放在一起

## 批量避重池

- 轮换轴：空间类型（咖啡馆 / 客厅 / 画廊 / 酒店 / 艺术馆）、机位远近（贴近家具 / 中景框景 / 长走廊）、动作（坐 / 站 / 走 / 靠）
- 高危重复骨架：
  - `窗边站姿 + 沙发坐姿 + 吧台近景`
  - `整批都在米白客厅里，没有外部空间变化`

## 负面约束

- 不要把都市松弛写成纯街拍
- 不要出现明显办公楼工位、会议室、普通商场过道
- 不要棚拍感太重
- 不要让背景只剩纯色墙和一把椅子
