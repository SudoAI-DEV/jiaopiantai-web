# 场景：自然田园

本文件定义 `自然田园` 场景的独立说明。重点不是城市街拍，而是“被园艺、建筑和生活方式整理过的庄园式自然 / 自然田园度假地”。当前批量任务同时吸收 `客户/{客户名}/场景/自然田园/` 与旧参考中的田园素材特征。

## 场景定位

- 核心气质：清新、浪漫、轻文艺、精致庄园感、度假式田园感
- 视觉重点：庄园草坪、碎石庭院桌景、白纱门廊、温室花房、藤蔓旧门、花境小径、池塘草坪、果园 / 茶园边、石木院屋、乡野别墅外立面、黄车花园边
- 必须保留：园艺修剪感、建筑或器物支点、家具 / 桌景 / 围栏 / 木栈道等人工维护痕迹
- 应避免：普通马路街景、普通公园草坪、粗糙田埂、廉价婚礼花门、只剩一片虚化绿背景没有空间层次

## 与相邻场景边界

- 不要写成 `都市街拍`：`自然田园` 必须有人造整理感、庄园感、度假物业感；如果一路写成街巷、咖啡馆、商业街区，就已经漂到 `都市街拍`
- 不要写成城市生活方式场景：`自然田园` 不是城市公寓、咖啡馆、精品酒店公区；即使有桌椅，也是花园 / 庭院 / 草坪里的户外生活区
- 不要写成 `艺术建筑`：`自然田园` 允许建筑，但建筑应该被植物、桌景、木石材料和柔和生活器物包围，而不是矿物感极简几何空间
- 不要写成 `海边艺术`：即使有白墙、露台或度假建筑，也不是海边棕榈 / 礁石 / 海风世界

## 关键词与影调

- 关键词：庄园草坪、白伞休闲区、碎石庭院下午茶、玻璃花房、白纱 veranda、藤蔓旧门、花境小径、池塘草坪、果园花树、茶园边坡、石木院屋、乡野 lodge、复古黄车
- 色盘：奶油白、鼠尾草绿、嫩绿、花瓣粉、雾黄、浅木色、碎石米色、池水青绿
- 光线：明亮午前柔光、午后暖光、叶影斑驳光、门廊柔光、水面反射补光
- 质感要求：背景必须清晰有层次，能看见花材、木石、围栏、水面或建筑边界，不能偷懒写成纯绿奶油虚化

## 单款执行规则

- 每款先从 `B01 ~ B20` 里选 1 个主场景方案，相邻兄弟 SKU 不要复用同一个编号
- `scene_01 ~ scene_08` 优先停留在同一庄园世界里，只允许切换同一物业内 2-3 个连通 micro-location，例如“花房正门 + 花房廊架 + 花房外花坛”，而不是“花房 + 池塘 + 黄车 + 石木院屋”平均混剪
- 若选了水岸、果园、茶园、乡路，也必须保留“被维护过”的线索，如栏杆、修剪边界、木栈道、院屋、围栏、桌椅、碎石路
- `复古黄车` 只能作为主场景世界里的点睛元素，不要 10 张都围着车拍

## 10 镜头编排建议

- 前 1-4 张要尽快立住主场景世界：至少给出 2 个不同的 micro-location，但它们必须来自同一庄园逻辑
- 坐姿 hero 更适合落在碎石庭院桌景、花园长椅、白纱门廊、廊架外摆、湖畔躺椅，而不是普通草坪席地坐
- 近景要保留花材、窗框、白纱、桌器、围栏、木扶手、水面反光、果树枝叶、石木门窗这些“精致田园锚点”
- 拼图镜头优先搭配花束、帽檐、茶具、桌面器物、藤编篮、木扶手、花器、黄车门把或栅栏细节

## 批量避重池

- 轮换轴：植物类型（花境 / 绿篱 / 果树 / 茶树 / 水边植物）、硬质结构（玻璃 / 石阶 / 碎石 / 栏杆 / 石木门窗 / 白栅栏）、水体与否、座具类型（藤椅 / 长凳 / 躺椅 / 长桌）、光线状态（午前清亮 / 午后暖光 / 门廊柔光 / 廊架条状影）
- 高危重复骨架：
  - `花房入口正面 + 花园小路背面 + 草地坐姿 + 藤蔓墙回眸`
  - `白纱门廊 + 藤蔓旧门 + 花园长椅` 在相邻 SKU 连续出现
  - `只换花草不换主环境`：看起来仍然像同一套温室 / 花园模板
- 当前批量优先分散：庄园草坪、碎石桌景、白纱门廊、花房、花境小径、池塘草坪、果园 / 茶园、石木院屋、乡野别墅、黄车花园边、湖畔草坪

## 负面约束

- 不要写成“山野旅拍版田园”，禁止荒山、野坡、粗糙林地、泥土地头、随意乡村公路
- 不要生成普通野草地、荒地或粗糙田埂；哪怕在乡野，也要有维护感和物业感
- 不要把田园自然写成婚礼花门、夸张花墙、球形气球、过度梦幻滤镜
- 不要把近景背景做成空白纯绿虚化，禁止明显奶油化景深偷懒
- 不要把场景写成纯建筑极简空间或纯都市外摆，那会漂到 `艺术建筑` / `都市松弛`
