# 场景：都市街拍

本文件定义 `都市街拍` 场景的独立说明。场景覆盖两个维度：**户外有质感的街区** 和 **城市精品内景空间**。核心是都市生活方式叙事——不管是街巷还是室内，都要有空间质感和生活气息，不是旅游打卡照或普通马路抓拍。

## 场景定位

- 核心气质：时髦、松弛、文艺、有生活方式叙事的都市空间（室内外兼有）
- 视觉重点分两线：
  - **街区线**：欧式文艺老街、鹅卵石街巷、铁艺阳台与藤蔓墙、复古咖啡馆外摆、精品买手店橱窗、老城广场、拱廊通道、旧石桥与运河边
  - **内景线**：设计师咖啡馆内部、独立书店、小型画廊展厅、买手店/Loft 内部、精品酒店公区（大堂、走廊、庭院）
- 必须保留：空间质感和纵深层次（室外看建筑立面，室内看材质和光线）
- 应避免：普通马路/停车场/商场大厅、杂乱人流街景、网红打卡摆拍、过于现代化的玻璃幕墙、廉价连锁店内部

## 与相邻场景边界

- 不要写成 `艺术建筑`：都市街拍的建筑是街区背景和生活容器，不是极简几何空间主角；内景空间是有烟火气的商业/文化空间，不是美术馆展厅
- 不要写成 `自然田园`：都市街拍可以有街边植物/藤蔓/室内绿植，但主体仍是城市环境
- 不要写成 `海边艺术`：即使有海边城市街区，主体仍是街巷叙事而非海岸度假
- 内景空间与 `艺术建筑` 的区别：这里的室内是有生活功能的商业空间（咖啡馆、书店、酒店），不是纯粹的建筑几何空间

## 关键词与影调

- 关键词（街区）：欧式老街、鹅卵石巷、铁艺阳台、藤蔓墙、咖啡馆外摆、精品店橱窗、拱廊、老城广场
- 关键词（内景）：设计师咖啡馆、独立书店、画廊展厅、买手店内部、Loft空间、精品酒店大堂、走廊光影、落地窗、木质吧台、书墙、陈列架
- 色盘（街区）：暖米石墙色、复古砖红、铁艺黑、藤蔓绿、粉墙柔粉、鹅卵石灰棕
- 色盘（内景）：暖木色、亚麻白、水泥灰、复古墨绿、黄铜色、旧皮革棕、低饱和奶油色
- 光线（街区）：午前清亮侧光、午后暖光、黄金时刻长影、街巷间的漫射柔光
- 光线（内景）：大窗自然光、吧台暖光、射灯聚光、走廊尽头逆光、落地窗侧光剪影

## 10 镜头编排建议

- 一组 10 镜头可以选择三种模式：
  - **纯街区模式**：10 张全在户外街区，前 1-4 张至少轮换窄巷 / 广场 / 咖啡馆 / 拱廊中的 3 类
  - **纯内景模式**：10 张全在室内精品空间，前 1-4 张至少轮换咖啡馆 / 书店 / 画廊 / 酒店中的 3 类
  - **混合模式（推荐）**：6-7 张户外 + 3-4 张内景，或反过来；要保持叙事连贯，例如"老街区漫步→走进街角咖啡馆→书店停留"这样的生活动线
- 全身镜头优先展示空间纵深和环境层次
- 近景保留可识别的空间元素（户外：铁艺栏杆、石墙纹理、门牌号；内景：书脊、咖啡杯、陈列物件）
- 拼图镜头应在统一的都市世界中取材，内外景在同一个街区/社区逻辑里

## 批量避重池

- 轮换轴：
  - 空间维度（户外街区 / 室内精品空间）—— 相邻 SKU 优先在这个维度上交替
  - 街区类型（老城窄巷 / 广场 / 咖啡馆 / 买手店 / 拱廊 / 水岸）
  - 内景类型（咖啡馆 / 书店 / 画廊 / 买手店 / 酒店 / vintage 空间）
  - 建筑立面 / 室内材质（石墙 / 砖墙 / 裸砖 / 水泥 / 木质 / 大理石）
  - 光线时段 / 光源（午前侧光 / 午后暖光 / 黄金时刻 / 大窗自然光 / 射灯暖光）
- 高危重复骨架：
  - `白墙正面站 + 咖啡馆坐姿 + 橱窗侧身 + 窄巷行走`
  - `一整批都在同一条街上只换角度`
  - `一整批内景都在咖啡馆，只换桌椅和角度`
  - `内景全是靠墙站/坐，没有利用空间纵深和道具`

## 硬性约束

### 通用
- 禁止杂乱人群和车辆（画面只保留模特一人）
- 禁止网红打卡式夸张摆拍
- 背景必须保留空间纹理和纵深，不要糊成色块

### 户外街区
- 禁止普通马路、停车场、无质感的现代建筑背景
- 优先选择有生活方式质感的街区，避免普通国内城市街景

### 城市内景
- 禁止连锁品牌门店内部（星巴克、宜家等有 logo 辨识度的空间）
- 禁止过于空旷的纯白空间（那是 `艺术建筑` 的领域）
- 禁止杂乱无设计感的普通店铺、办公室、居民室内
- 室内必须有可辨识的空间功能和生活道具（书、咖啡器具、陈列品、家具等），不能只是一面墙
- 室内光线必须有层次（窗光+灯光、明暗区域），禁止全均匀打光的"证件照"感
