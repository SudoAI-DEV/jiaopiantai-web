# 七大风格模板定义

本文件现在作为 **风格总索引 + 通用规则 + 批量避重入口**。  
每个风格的具体场景家族、micro-location、光线和负面约束，已经拆到独立 markdown：

- `references/styles/style-a-seaside-art.md`
- `references/styles/style-b-country-garden.md`
- `references/styles/style-c-urban-relaxed.md`
- `references/styles/style-d-modern-tech.md`
- `references/styles/style-e-architectural-editorial.md`
- `references/styles/style-f-outdoor-street.md`
- `references/styles/style-g-refined-nature.md`

`styled-clothing-shoot` 仍以 `SKILL.md` 中的 20 镜头规则为准；本文件负责帮助 Agent 快速判断风格边界、批量避重与优先阅读入口。户外天气必须跟随真实参考图，不要把所有外景都统一成烈日或阴天。

通用补充规则：
- 默认清场，不要在背景里生成路人、游客、外摆顾客或其他抢镜人物。
- 连衣裙、半裙、裤装优先保留口袋结构，并在合适镜头中随机安排自然手插口袋或拇指勾口袋边动作。
- 所有服装默认有内衬且不透；除非客户明确要求透明设计，否则不要生成透视薄纱感。
- 街景、田园、海景、艺术馆和建筑场景都要有更高端的空间材质与环境叙事，不要落成普通游客照。

## 批量任务防重总规则（适用于 `2026_春夏_第二批` 等同季多 SKU）

- 先看同一 `season + 风格名` 目录下最近 4-8 个兄弟 SKU 已完成的 `生成/batch_*/scenes.json`，再写当前款。
- 当前款必须避开最近 3 个兄弟 SKU 的高频 `scene_family + micro_location` 组合，尤其是前 1-10 张全身镜头的 hero 场景骨架。
- 不要把“风格模板表”理解成固定答案；它只是风格边界，不是每一款都要照抄同一组背景顺序。
- 若某款衣服因材质、长度或客户要求必须使用某个已出现过的主环境，至少轮换剩余 5 个全身镜头与 3 个近景落点。
- 推荐在 `scenes.json` 里填写 `scene_family`、`micro_location`、`diversity_reason`，方便下一款继续避重。

## 当前第二批重点风格轮换池

以下三组是当前仓库里 `客户/花姐&寅寅/2026_春夏_第二批/` 的重点风格。后续批量跑图时，优先按这些轮换池分散环境。

### 田园自然（B）批量轮换池

- 可轮换主环境：温室花房入口、白纱门廊、庄园石阶、藤蔓石墙、修剪花坛边、林荫石板路、花园长椅、露台下午茶区、玻璃花房内侧、花圃木门边
- 高危重复骨架：`花房入口正面 + 花园小路背面 + 草地坐姿 + 藤蔓墙回眸`
- 轮换优先轴：花材密度、硬质建筑元素（石阶 / 门框 / 栅栏）、座具类型（藤椅 / 长凳 / 石台）、光线状态（午前柔光 / 午后斑驳光）

### 艺术建筑（E）批量轮换池

- 可轮换主环境：弧形长廊、门洞前厅、清水混凝土楼梯、夯土长墙、树影中庭、微水泥平台、光井边、连桥廊道、转角台阶、艺术馆露台
- 高危重复骨架：`门洞正面 + 楼梯背面 + 长廊行走 + 台阶坐姿`
- 轮换优先轴：几何形状（弧 / 直 / 洞口）、材质（夯土 / 微水泥 / 混凝土 / 石材）、光源状态（硬朗日照 / 天窗柔光 / 树影切割）
- 执行硬规则：
  - 先从 `style-e-architectural-editorial.md` 的 20 个具体建筑场景方案里选 1 个，再开始写当前 SKU
  - 相邻兄弟 SKU 不要共享同一个建筑场景方案编号
  - `scene_01 ~ scene_18` 必须停留在同一个建筑世界里，只允许切换同一建筑 / 同一建筑群内部逻辑连通的 2-3 个 micro-location
  - `scene_01 ~ scene_18` 必须写满 `pose / lighting / background / model_direction / color_tone`

### 大自然（G）批量轮换池

- 可轮换主环境：芦苇荡边缘、花海步道、草原坡地、高草坡顶、梯田观景点、湿地栈道、山野露台、民宿木平台、溪谷草坡、草甸山脊
- 高危重复骨架：`普通草地背面 + 普通稻田正面 + 压暗黄昏站立 + 露台坐姿`
- 轮换优先轴：地貌类型（芦苇 / 花海 / 草原 / 梯田 / 溪谷 / 露台）、景观精致度（花材密度 / 芦苇层次 / 草坡起伏 / 栈道结构）、阳光状态（晴天午后 / 金色侧光 / 明亮多云但有太阳方向）
- 硬性禁令：禁止明显奶油化虚焦背景；禁止无日照的阴暗山野；禁止普通公园草地、路边树林、廉价郊游照式自然背景

| 编号 | 镜头类型 | 说明 |
|------|---------|------|
| 1 | 整背面 | 全身背面，展示背部版型与线条 |
| 2 | 整正面（全身） | 全身正面，头到脚完整展示 |
| 3 | 全身背面行走 | 背面动态，自然步伐或回眸 |
| 4 | 细节图 | 上半身或局部特写，展示面料、纹理、配饰 |
| 5 | 坐姿 | 坐在场景道具上的自然坐姿 |
| 6 | 其他（靠墙 / 道具） | 靠墙、拿道具、互动等变化动作 |

## 相邻风格边界

- `田园自然 (B)`：偏花园、庭院、藤蔓、白纱、复古家具，是被整理过的人造田园。
- `大自然 (G)`：偏草海、梯田、山景、稻田、乡村露台、树林、黄昏田野，是更大的山野旅行环境。
- `都市松弛 (C)`：偏室内或室内外连通的生活方式空间。
- `室外街拍 (F)`：偏门店外立面、人行道、咖啡馆外摆、街角、招牌和橱窗。
- `现代科技 (D)`：偏冷调玻璃 / 金属 / 灯带 / 亚克力 / 展厅。
- `艺术建筑 (E)`：偏暖调夯土 / 微水泥 / 清水混凝土 / 弧墙 / 楼梯 / 几何洞口 / 树影。

## 独立风格文件使用方法

- 在开始写某一款 `scenes.json` 前，先看本文件里的通用规则和当前风格的批量轮换池。
- 然后必须继续阅读对应风格的独立 markdown，再决定主场景家族与 micro-location。
- 如果是 `2026_春夏_第二批` 这种连续批量任务，优先把独立风格 markdown 里的“高危重复骨架”和“场景家族”当成避重依据。

## 风格 B：田园自然

**关键词**：花园、庭院、藤蔓、白纱、草地、小径、温室花房、庄园露台、复古家具、清新、浪漫、高级

**氛围**：清新浪漫的田园诗意，自然光下柔和温暖，带有文艺与生活气息。重点是被整理过的花园感与庄园式精致度。

**光线与色盘**：明亮暖光、柔和日照、树影斑驳、奶油白与嫩绿色调。

**配饰方向**：耳环、草帽、花束、藤编篮、丝巾、帆布包、手提包、咖啡杯、高跟鞋

| 场景 | scene_type | pose | background | lighting |
|------|-----------|------|------------|----------|

| 2-整正面 | 户外 / 半室内 | 正面全身站在花丛前或温室花房入口，轻触花瓣，神情温柔 | 藤蔓、白纱、花园拱门、玻璃花房、复古桌椅 | 午前自然光，花叶漫反射补光 |
| 3-全身背面行走 | 户外 | 穿过林荫小径或庄园露台花园的背面行走，裙摆轻摆 | 石板路、灌木、修剪整齐的绿篱、树影和庭院层次 | 斑驳树影，逆光透过叶片 |
| 4-细节图 | 户外 / 半室内 | 上半身侧面特写，低头闻花束或整理丝巾 | 虚化花丛、绿叶、白纱帘、温室窗框 | 侧面柔光，肤色温润 |
| 5-坐姿 | 户外 | 坐在庄园露台藤椅、花园长凳或草地毯上，姿态自然 | 草坪、石桌、花器、小桌道具、精致餐具 | 午后暖光，草地与石材反射柔和补光 |
| 6-其他 | 户外 | 倚靠旧木门框、藤蔓石墙或花园露台栏杆，手持草帽或花束回眸 | 爬满藤蔓的石墙、旧门、花盆、露台立柱 | 午后阳光穿过藤蔓形成层次光影 |

## 风格 E：艺术建筑

**关键词**：夯土墙、微水泥、清水混凝土、弧形楼梯、长廊、几何洞口、树影、艺术馆、建筑空间、极简时装

**氛围**：从新增参考图提炼出的建筑空间时装感。环境不是未来科技，而是暖调矿物材质、艺术馆空间与几何结构形成的静谧大片感。

**光线与色盘**：暖米色、沙色、咖色、奶油白、黑白低饱和；多用清晰日照、硬朗投影、树影或窗洞光束。

**配饰方向**：几何耳环、结构感包、窄檐草帽、踝靴、尖头鞋、简约手镯

**批量避重提醒**：不要让艺术建筑连续多款都固定成“弧墙门洞 + 楼梯 + 长廊”。应轮换树影中庭、光井、露台、连桥、转角台阶、夯土长墙等不同几何关系。

| 场景 | scene_type | pose | background | lighting |
|------|-----------|------|------------|----------|
| 1-整背面 | 半户外 | 模特沿弧形长廊或坡道走开，肩线舒展 | 暖米色矿物墙面、长廊、洞口露出一小块天空 | 正午或午后硬朗侧光，几何阴影清晰 |

## 风格 F：室外街拍

**关键词**：门店外立面、咖啡馆、露天餐厅、橱窗、人行道、街角、路缘、漂亮建筑、画廊入口、树影、通勤、抓拍、干净背景、少人街景

**氛围**：从新增参考图提炼出的城市外景穿搭感。重点是门店外拍、高级街角与外摆餐饮空间的生活感，不是棚拍，也不是单一复制的门店模板。背景必须干净克制，像被提前清过场的品牌街拍，不要出现拥挤路人和脏乱街景。

**光线与色盘**：午后斜阳、街边树荫、明亮阴天、橱窗反光；色盘以城市中性色为主，允许少量招牌或墙面跳色，但要压低视觉噪音，保持背景简洁。

**场景筛选规则**：
- 优先：精品门店外立面、安静街角、简洁橱窗、漂亮建筑转角、非高峰时段露天餐厅、画廊入口。
- 避免：拥挤步行街、游客打卡点、杂乱停车路边、密集招牌区、背景人流明显的商圈主街。
- 构图上优先让门框、墙面、窗景、雨棚、树影、栏杆做背景切割，减少远处行人和杂物进入画面。

**配饰方向**：墨镜、咖啡杯、托特包、腋下包、帽子、耳环、乐福鞋、踝靴、高跟鞋

## 风格 G：大自然

**关键词**：芦苇荡、花海、草原坡地、高草草坡、梯田观景点、湿地栈道、山野露台、溪谷草坡、旅行、明亮阳光

**氛围**：从新增参考图提炼出的精致山野旅拍感。背景本身要高级、干净、层次明确，像品牌 lookbook 在自然环境里的大片，而不是普通郊游照。环境可以占比更大，但服装始终是主角。

**光线与色盘**：草木绿、芦苇金、花海粉白、奶油白、草原金绿；优先晴天午后、金色侧光、明亮多云但有太阳方向的通透山野光。不要阴暗灰闷、不要没太阳的薄暮、不要压暗背景。

**配饰方向**：草帽、编织篮、披肩、耳环、编织包、长靴、平底凉鞋、高跟鞋

**批量避重提醒**：不要把大自然持续压成“普通草地 + 普通稻田 + 黄昏田野”三件套。应主动切换芦苇荡、花海、草原坡地、湿地栈道、梯田观景位、山野露台、溪谷草坡等不同地貌。

**硬性约束**：
- 背景不得出现明显大虚化、奶油化假景深；自然环境纹理和远近层次必须保留
- 禁止没有阳光的阴暗背景、灰闷薄暮、压暗天空和过重阴影
- 优先“被挑选过的高级自然场景”，不要普通路边树林、普通空草地、廉价旅游景点感背景
