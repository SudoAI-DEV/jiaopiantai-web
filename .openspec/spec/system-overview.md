# 系统概述

## 项目背景

蕉片台 (Jiaopiantai) 是一个 AI 商品图片生成服务平台，为服装店、饰品店等有商品拍摄需求的中小商家提供低成本、高效率的商品图片生成服务。

## 核心价值主张

- **低成本**: 客户只需上传产品原始照片，AI 自动生成专业商品图
- **高质量**: 内部人工审核把关，确保交付图片质量
- **高效率**: 简化传统拍摄流程，省去模特、拍摄、后期等环节

## 技术栈

- **前端**: Next.js 14+ (App Router), React 18+, TypeScript, Tailwind CSS, shadcn/ui
- **后端**: Next.js API Routes, Drizzle ORM, PostgreSQL
- **认证**: Better Auth
- **存储**: 阿里云 OSS
- **状态管理**: Zustand

## 用户角色

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 客户 (Customer) | 注册商家用户 | 新建产品、上传图片、填写需求、消耗点数、查看成品图 |
| 管理员 (Admin) | 内部运营人员 | 全部客户管理、点数充值、图片审核、交付确认 |

## 核心业务流程

```
客户注册 → 线下沟通购买 → 管理员后台充值点数 → 新建产品 → 上传图片/填写需求 → 消耗点数提交 →
AI 异步生成（约20张） → 管理员审核筛选 → 交付客户（约6张精选）
```

## 服务模式

本平台是**半自动化的服务交付平台**，不是自助型 AI 生成平台：
- 客户端：提交需求、查看成品
- AI 端：异步拉取任务并生成图片
- 管理端：审核图片、管理客户、管理配额
