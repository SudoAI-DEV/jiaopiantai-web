## 1. 快速开始

### 1.1 注册账号
1. 访问 https://jiaopiantai.com
2. 点击「注册」按钮
3. 填写邮箱、密码、店铺名称、联系电话
4. 点击「注册」完成

### 1.2 登录
1. 访问 https://jiaopiantai.com/login
2. 输入邮箱和密码
3. 点击「登录」

## 2. 客户操作指南

### 2.1 仪表盘
登录后进入仪表盘，可查看：
- 当前点数余额
- 产品列表
- 快捷操作入口

### 2.2 新建产品
1. 点击「新建产品」或「+」按钮
2. 填写产品信息：
   - 产品名称（必填）
   - 产品类目（必填）
   - 产品描述（选填）
   - 拍摄需求（选填）
   - 风格偏好（选填）
   - 特别说明（选填）
3. 上传产品图片（支持 JPG/PNG/WEBP，单个文件最大 10MB）
4. 点击「保存为草稿」或「提交」

### 2.3 提交产品
1. 在产品列表中找到要提交的产品
2. 点击「提交」按钮
3. 确认消耗 1 点数
4. 等待 AI 生成（约 5 分钟）
5. 可在产品详情页查看实时进度

### 2.4 查看成品
1. 产品状态变为「待核对」时
2. 进入产品详情页查看生成图片
3. 支持以下操作：
   - 单张图片下载
   - 批量下载 ZIP
   - 提交反馈

### 2.5 提交反馈
1. 在成品查看页面
2. 选择反馈类型：
   - 满意
   - 需要修改
   - 质量问题
   - 风格不符
   - 其他
3. 填写详细描述
4. 点击「提交反馈」

## 3. 管理员操作指南

### 3.1 仪表盘
管理员登录后进入仪表盘，可查看：
- 平台概览统计
- 待处理任务列表
- 今日数据

### 3.2 客户管理
1. 进入「客户管理」
2. 可查看所有客户列表
3. 支持搜索和筛选
4. 点击客户可查看详情
5. 可进行充值点数操作

### 3.3 点数充值
1. 进入客户详情页
2. 点击「充值点数」
3. 输入充值数量
4. 填写备注（选填）
5. 点击「确认充值」

### 3.4 产品审核
1. 进入「产品管理」
2. 找到待审核的产品
3. 点击「审核」进入审核页面
4. 查看 AI 生成的图片
5. 对每张图片进行操作：
   - ✅ 通过
   - ❌ 淘汰
   - 🔄 重新生成
6. 可批量选择图片进行操作
7. 通过数量达到期望后，点击「确认交付」

### 3.5 快捷键
在审核页面可使用快捷键：
- `←` / `→`：上一张/下一张图片
- `J`：通过当前图片
- `K`：淘汰当前图片
- `A`：重新生成当前图片

## 4. 常见问题

### 4.1 点数相关
**Q: 点数有什么用？**
A: 每提交一个产品进行 AI 生成，需要消耗 1 点数。

**Q: 点数可以退款吗？**
A: 如果 AI 生成失败，点数会自动退还。如果客户反馈需要修改，管理员会酌情处理。

**Q: 如何充值点数？**
A:请联系客服或管理员进行充值。

### 4.2 图片相关
**Q: 支持哪些图片格式？**
A: 支持 JPG、PNG、WEBP 格式。

**Q: 图片文件大小限制？**
A: 单个文件最大 10MB。

**Q: 可以上传多少张原始图片？**
A: 建议上传 3-5 张不同角度的产品图片。

### 4.3 生成相关
**Q: AI 生成需要多长时间？**
A: 一般 5-10 分钟完成生成。

**Q: 可以生成多少张图片？**
A: 每次生成约 20 张图片，管理员审核后会交付 6 张精选。

**Q: 对生成结果不满意怎么办？**
A: 可以在成品页面提交反馈，管理员会安排重新生成。

### 4.4 账户相关
**Q: 忘记密码怎么办？**
A: 在登录页面点击「忘记密码」进行找回。

**Q: 如何联系客服？**
A: 微信号：xxx（请咨询客服）
